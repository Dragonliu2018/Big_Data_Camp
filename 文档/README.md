# 文档介绍

　　本模块为主要存放一些平常的文档+过程源码等，主要目的是使15天的大数据分析训练营留下足迹，最后课程需要提交这部分的材料。

文件结构如下：

* **01_需求说明书**(存放需求说明书的各个版本及模板)
* **02_项目计划书**(存放项目计划书的各个版本及模板)
* **03_project**(存放project的各个版本及模板、项目实施细则)
* **04_源代码**(存放各个版本的源代码)
  * **01.数据采集**(存放各个版本的链家网全国房源爬虫代码)
  * **02.数据存储**(存放各个版本的数据存储代码)
  * **03.数据分析**(存放各个版本的数据分析代码)
  * **04.数据可视化**(存放各个版本的数据可视化代码)
* 中间有待补充
* **10_笔记**(存放课程笔记，目前主要在notion)
* **11_会议记录**(存放会议记录，目前主要在notion)
* **12_例会手稿**(存放例会手稿，方便例会展示)
* **13_进程存档**(存放讨论出来的好的想法，如前端的草图、用例图、类图、时序图等等)

* **14_技术文档**(存放一些技术文档，方便大家查找)
* **15_参考资料**(存放pdf等书籍及上往年的项目，方便大家查找)
* **16_课程PPT**
